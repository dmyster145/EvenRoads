Getting Started

Welcome

EvenRoads is a road crossing game for Even Realities G2. Move left or right, hop up, and cross traffic to reach the goal row.

Objective

Start on the bottom row and reach the top row without crashing. New games spawn at a random bottom-row column. After each successful crossing, you respawn on the bottom row in the same column.

Launch Behavior

  • A new game starts immediately
  • Only best score is restored
  • After a crash, double tap to start a new game at a random bottom-row column

What This Covers

This guide reflects the current build: controls, rules, hazards, and progression.

Board Layout

Playfield Dimensions

  • Width: 28 columns
  • Height: 9 rows
  • Top row: Goal lane
  • Bottom row: Start lane
  • Middle rows: Road lanes with moving traffic

Board Glyphs

Glyph Meaning
▲Player while alive
※Player marker on death
◈, ▷, ◁Road traffic (arrow variants indicate flow direction)
▒Open road cell
▩Solid blocker (impassable)
□Bridge safe cell and part of goal/start row patterns
Note: Bridge cells are always safe, even if traffic passes through the same cell.
Controls

Use these inputs:

Scroll Up Move right one column.
Scroll Down Move left one column.
Tap Hop up one row.
Double Tap Hop up while running. Restart after a crash.

Gameplay Flow

Basic Loop

  1. Align with safe openings using left/right movement
  2. Tap to hop up one row at a time
  3. Avoid traffic and solid blockers
  4. Reach the top row to score and level up

Lane Motion

  • Road lanes shift over time while state is ALIVE
  • Adjacent road rows flow in opposite directions
  • Higher levels speed up lane updates
  • Traffic wraps around screen edges
Rules

Movement Rules

  • Left/right moves clamp at board edges
  • Up movement cannot go above the goal row
  • There is no down action
  • You cannot move into solid blocker cells

Collision Rules

  • Touching traffic on a non-bridge road cell causes a crash.
  • Bridge cells ignore traffic collision
  • The HUD state label blinks CRASHED! while crashed

Restart

  • When crashed, only double tap starts a new game
Tip: Use direction markers (▷/◁) to predict where gaps will be next tick.
Scoring & Progression

Scoring

  • Each top-row crossing adds +1 score
  • Best updates when current score exceeds prior best

Leveling

  • Each successful crossing increases level by 1
  • Each new level regenerates traffic, blockers, and bridges
  • Tick interval decreases with level down to a minimum floor

On Goal Reach

  • Message updates to Crossed! Level #
  • Player respawns on the bottom row at the same column used to cross
  • The game continues immediately at higher difficulty
Status & Feedback

Scoreboard Line

The status line shows:

  • Score: ##
  • Best: ##
  • State: ALIVE / CRASHED!
  • Crossed! Level # appears after crossing until you leave the home row
Persistence

What Is Saved

  • Only best score is persisted
  • It is stored in browser localStorage as evenroads.bestScore
  • Use Reset Scores to clear current score and best score, then start a new game

What Is Not Saved

  • Current game position, score, level, and board layout are not resumed
  • Each launch starts a fresh game
Tips & Notes

Practical Play Tips

  • Move horizontally early to line up with bridge cells.
  • On higher levels, commit sooner because lanes update faster.
  • If a hop queues, watch lane motion before your next input.
  • Read alternating lane flow early and time hops one row ahead.
